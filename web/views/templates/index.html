nexus.io

Keep the control of your home devices.
Get Started
npm install nexus.io

Nexus.io is an easy-to-use node.js module to references make your computers and embedded boards registrable online and make them detectable by other devices on the same local network.

Fast IoT

10 lines to turn your desktop app into a connected object and let your devices interact between each others.

Security

The best security online is offline. With nexus.io, the detection is online but the communication between devices is only possible through your private network.

Instantaneity

Nexus.io uses universal protocol websockets. So you can detect in real-time the devices in your private network without any latency.
